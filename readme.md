# colrz

bloat free terminal colors

## usage

``` javascript
require('colrz');

console.log('red underlined text'.red.underline);
```

## disable

set `NO_COLOR` in $env or use `--no-color` argument

## force

set `FORCE_COLOR` in $env or use `--force-color` argument
